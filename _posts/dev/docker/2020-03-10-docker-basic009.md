## 1. 도커 네트워크 확인

- 도커 네트워크 목록 확인
``sh 
docker network ls
``

- 도커 네트워크 자세한 정보 확인

```sh
docker network inspect bridge
```
## 2. 사용자 정의 브리지 네트워크

- 사용자 정의 브리지를 새로 생성 

```sh 
docker network create --driver bridge whms-bridge
```

- 사용자 정의 브리지 옵션추가 생성

```sh 
docker network create --driver bridge \
--subnet=172.72.0.0/16 \
--ip-range=172.72.0.0/24 \
--gateway=172.72.0.1 \
whms-custom-bridge
```
